# Kutt

Simple API wrapper for [kutt.it](https://kutt.it)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     kutt:
       github: watzon/kutt
   ```

2. Run `shards install`
